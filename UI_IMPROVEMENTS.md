## Usage

Just start the web UI as before:

```bash
./start_web.sh
# Or
./bin/web -master localhost:50051 -port 8080
```

Then open **http://localhost:8080** in your browser!

## Features in Action

1. **Submit a task** - Watch the button animate on submit
2. **Filter tasks** - Click filter buttons to see different states
3. **Watch stats** - See real-time updates in stat cards
4. **Hover effects** - Hover over cards to see animations
5. **Auto-refresh** - Tasks update every 2 seconds automatically

Enjoy the new UI! 🚀
